前言

上次我们配置好了环境，这次我来教教大家怎么把你的博客（Hexo）怎么上传到Github空间！

正文

1/0 注册GitHub

首先，你得有一个GitHub账号，账号需要邮箱 （如果你有QQ，就可以直接用: 你的QQ号+@qq.com)，然后在进入https://github.com/，点击Sign up ，注册一个账号。

1/1 注册仓库

点击Create repository 创建一个仓库。

在Repository name输入你的名称+.github.io。

在下面的Initialize this repository with中选择Add a README file。

现在的基本配置都已经准备好了，开始上传！

2/0 获取密钥

打开git bash,输入

ssh-keygen -t rsa -C “your_email@youremail.com“

在"your_email@youremail.com"换上你的邮箱。

不管出现什么信息，你只需要回车就可以了。

然后，再输入

 
						cat ~/.ssh/id_rsa.pub

会输出你的密钥，我们复制输出信息就行了

然后点击头像，点击Settings--SSH·····--New SSH Key

粘贴你复制的秘钥。

然后打开git bash

执行：


                        ssh -T git@github.com

会提示你下面这个，输入yes回车就行了

Are you sure you want to continue connecting (yes/no/[fingerprint])?

打开项目，点击绿色按钮，点击SSH，再点复制

打开站点配置文件(_config.yml)

修改deploy信息。


                        deploy:

                         type: git

						 repo: 你复制的地址

                         branch: master

在博客根目录下打开git bash

分别执行下面的命令：

your_github_name是你github的名字

your_github_email是你注册github时的邮箱


                        git config --global user.name "your_github_name"

						git config --global user.email "your_github_email"

然后安装上传插件


                        cnpm install hexo-deployer-git --save

2/1 上传！

执行


                        hexo g -d

或是


                        hexo d

就部署好了你的博客了！

这里要注意一点：Github的分支变成了main，所以要修改。


                        branch: main

后记

输入你的网址，就可以看到网站了！
